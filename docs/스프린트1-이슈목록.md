## GitHub에 이슈 등록하는 방법

1. 레포지토리 → **Issues** 탭 → **New issue**
2. 버그 리포트 / 기능 요청 중 **기능 요청** 템플릿 선택
3. 제목: `[feat] #N. 이슈 제목` 형식으로 작성
4. 레이블: `planning` / `design` / `frontend` / `chore` 중 선택
5. 마일스톤: `스프린트 1 — 기획·설계` 선택
6. 담당자(Assignee) 지정 후 Submit
7. Projects 칸반 보드에서 **Backlog** 컬럼으로 이동
