## US-01. 공통 내비게이션

### US-01-1. GNB로 페이지 이동

**As a** 모든 방문자(P1~P6),
**I want to** 어느 페이지에서든 헤더의 메뉴(GNB)를 클릭해 원하는 페이지로 이동하고 싶다,
**So that** 사이트 구조를 쉽게 파악하고 원하는 정보에 빠르게 접근할 수 있다.

**Acceptance Criteria**
- [ ] 헤더에 로고, 회사소개, 사업분야, IR, 문의하기 메뉴가 표시된다
- [ ] 각 메뉴 클릭 시 해당 페이지로 이동한다
- [ ] 현재 활성화된 메뉴 항목은 시각적으로 구분(색상, 밑줄 등)된다
- [ ] 스크롤을 내려도 헤더가 화면 상단에 고정(sticky)된다
- [ ] 로고 클릭 시 메인(/)으로 이동한다

---

### US-01-2. 모바일에서 햄버거 메뉴 사용

**As a** 스마트폰으로 접속하는 방문자(P1~P6),
**I want to** 햄버거 아이콘을 탭해 전체 메뉴를 열고 닫고 싶다,
**So that** 작은 화면에서도 모든 메뉴에 접근할 수 있다.

**Acceptance Criteria**
- [ ] 767px 이하 화면에서 GNB 메뉴가 숨겨지고 햄버거 아이콘이 표시된다
- [ ] 햄버거 아이콘 탭 시 전체 메뉴 드로어/오버레이가 열린다
- [ ] 메뉴 항목 탭 시 해당 페이지로 이동하고 메뉴가 닫힌다
- [ ] 메뉴 외부 영역 탭 또는 X 버튼으로 메뉴를 닫을 수 있다
- [ ] 메뉴 열림/닫힘 시 키보드 포커스가 올바르게 이동한다

---

### US-01-3. 푸터에서 추가 정보 확인

**As a** 모든 방문자(P1~P6),
**I want to** 페이지 하단 푸터에서 회사 기본 정보와 관련 링크를 확인하고 싶다,
**So that** 주소, 연락처, 패밀리사이트를 빠르게 찾을 수 있다.

**Acceptance Criteria**
- [ ] 회사명, 주소, 대표번호, 사업자등록번호가 표시된다
- [ ] 저작권 표시(© GC메디아이)가 포함된다
- [ ] 패밀리사이트(의사랑, 유팜, UBIST, 똑닥, 미소몰) 링크가 제공된다
- [ ] 외부 링크는 새 탭(`target="_blank"`)으로 열린다
- [ ] 개인정보처리방침 링크가 포함되며, `/privacy` 페이지로 이동한다

## US-02. 메인 페이지 (/)

### US-02-1. 브랜드 비전 확인

**As a** 처음 방문하는 잠재 고객 또는 파트너(P1, P3),
**I want to** 첫 화면에서 GC메디아이가 어떤 회사인지 한눈에 파악하고 싶다,
**So that** 이 사이트에서 내가 찾는 정보를 얻을 수 있는지 빠르게 판단할 수 있다.

**Acceptance Criteria**
- [ ] 히어로 섹션에 "Beyond EMR, Toward Medical OS" 메시지가 명확하게 표시된다
- [ ] GC메디아이의 핵심 가치를 설명하는 서브 카피가 포함된다
- [ ] 회사소개, 사업분야로 이동하는 CTA 버튼이 1개 이상 표시된다
- [ ] 히어로 이미지 또는 비주얼이 의료·AI 테마를 반영한다
- [ ] 모바일/태블릿에서 히어로 레이아웃이 깨지지 않는다

---

### US-02-2. 핵심 역량 수치 확인

**As a** 투자자 또는 파트너(P2, P3),
**I want to** GC메디아이의 규모와 기술 역량을 나타내는 수치를 메인에서 확인하고 싶다,
**So that** 협력 또는 투자 가치를 신속하게 가늠할 수 있다.

**Acceptance Criteria**
- [ ] 의료기관 네트워크 25,000처+, 누적 의료 데이터 2억 건+, H100 기반 Medical LLM 수치가 표시된다
- [ ] 각 수치에 설명 레이블이 함께 표시된다
- [ ] 수치 섹션은 데스크톱·모바일 모두에서 가독성 있게 배치된다

---

### US-02-3. 사업분야 요약 탐색

**As a** 잠재 고객 또는 파트너(P1, P3),
**I want to** 메인 페이지에서 GC메디아이의 주요 사업 영역을 한눈에 파악하고 싶다,
**So that** 관심 있는 서비스를 찾아 상세 페이지로 이동할 수 있다.

**Acceptance Criteria**
- [ ] 메디컬 인프라, 데이터·마케팅, 플랫폼, 유통·의료기기 4개 영역 카드가 표시된다
- [ ] 각 카드에 영역명, 대표 서비스명(의사랑/유팜 등), 한 줄 설명이 포함된다
- [ ] 카드 클릭 시 사업분야(/business) 페이지 또는 해당 섹션으로 이동한다

---

### US-02-5. 메인에서 IR 주요 지표 미리보기 확인

**As a** 투자자(P2),
**I want to** 메인 페이지에서 GC메디아이의 주요 재무 지표를 간략히 확인하고 싶다,
**So that** IR 페이지로 이동하기 전 핵심 투자 정보를 즉시 파악할 수 있다.

**Acceptance Criteria**
- [ ] 메인 페이지에 최신 연도 기준 1~2개 핵심 재무 지표(예: 매출, 영업이익)가 정적으로 표시된다
- [ ] 지표 아래에 기준일(예: "2024년 기준")이 명시된다
- [ ] "IR 바로가기" 또는 유사한 CTA 링크 클릭 시 `/ir` 페이지로 이동한다

---

### US-02-4. 최신 뉴스 미리보기 확인

**As a** 언론 관계자 또는 일반 방문자(P4, P6),
**I want to** 메인 페이지에서 최신 보도자료를 확인하고 싶다,
**So that** 클릭 한 번으로 뉴스 상세 내용에 접근할 수 있다.

**Acceptance Criteria**
- [ ] 최근 뉴스 3~5건의 제목과 날짜가 표시된다
- [ ] 각 뉴스 항목 클릭 시 IR 내 NEWS 섹션 또는 해당 기사로 이동한다
- [ ] "더 보기" 링크 클릭 시 IR(/ir) 페이지 NEWS 섹션으로 이동한다

## US-03. 회사소개 (/about)

### US-03-1. 기업 스토리 이해

**As a** 구직자 또는 일반 방문자(P5, P6),
**I want to** GC메디아이가 어떤 과정을 거쳐 현재에 이르렀는지 이야기로 이해하고 싶다,
**So that** 기업의 방향성과 가치관을 파악하고 관심 여부를 결정할 수 있다.

**Acceptance Criteria**
- [ ] 현재(의료 IT 인프라) → 전환(AI 중심 전략) → 미래(Medical OS 비전) 3단 내러티브가 구성된다
- [ ] 각 단계가 시각적으로 구분되어 순서대로 읽힌다
- [ ] 텍스트만으로 내러티브가 전달되며 이미지가 없어도 의미가 통한다

---

### US-03-2. 회사 연혁 탐색

**As a** 투자자 또는 파트너(P2, P3),
**I want to** GC메디아이의 주요 이정표를 연도순으로 확인하고 싶다,
**So that** 기업의 성장 궤적과 안정성을 평가할 수 있다.

**Acceptance Criteria**
- [ ] 연도별 주요 사건이 타임라인 형태로 표시된다
- [ ] 각 항목에 연도, 주요 사건 설명이 포함된다
- [ ] 가장 최신 연도가 상단 또는 끝에 위치한다
- [ ] 모바일에서 타임라인이 세로 방향으로 자연스럽게 표시된다

---

### US-03-3. 가족사·관계사 파악

**As a** 파트너 담당자(P3),
**I want to** GC메디아이의 계열사 및 관계사 현황을 확인하고 싶다,
**So that** 협력 가능한 사업 영역의 범위를 파악할 수 있다.

**Acceptance Criteria**
- [ ] 가족사·관계사 로고와 이름, 한 줄 설명이 표시된다
- [ ] 각 사의 공식 사이트 링크가 제공된다 (새 탭 이동)
- [ ] 로고 이미지에 alt 텍스트가 설정된다

**As a** 방문 예정 파트너 또는 구직자(P3, P5),

**So that** 방문 전 경로를 미리 파악할 수 있다.

**Acceptance Criteria**

- [ ] 지도 로드 실패 시 주소 텍스트가 대체 콘텐츠로 표시된다

## US-05. IR (/ir)

### US-05-1. 재무 정보 확인

**As a** 투자자(P2),
**I want to** GC메디아이의 최근 재무 현황을 확인하고 싶다,
**So that** 투자 판단에 필요한 기초 정보를 얻을 수 있다.

**Acceptance Criteria**
- [ ] 최근 연도의 주요 재무 지표(매출, 영업이익 등)가 표시된다
- [ ] 데이터 출처와 기준일이 명시된다
- [ ] 표 또는 카드 형태로 가독성 있게 표현된다

---

### US-05-2. 공시자료 접근

**As a** 투자자(P2),
**I want to** 공시자료 목록을 확인하고 원문에 접근하고 싶다,
**So that** 공식 정보를 직접 검토할 수 있다.

**Acceptance Criteria**
- [ ] 공시 제목, 공시 일자가 목록 형태로 표시된다
- [ ] 각 공시 항목에 원문 링크(DART 또는 거래소)가 제공된다
- [ ] 링크는 새 탭으로 열린다
- [ ] 목록이 날짜 내림차순으로 정렬된다

---

### US-05-3. 주주공고 확인

**As a** 주주(P2),
**I want to** 주주총회 및 배당 관련 공고를 확인하고 싶다,
**So that** 주주 권리 행사 일정을 파악할 수 있다.

**Acceptance Criteria**
- [ ] 주주공고 섹션에 최신 공고 내용이 표시된다
- [ ] 공고 제목, 날짜, 내용 요약이 포함된다
- [ ] 공고 원문 링크 또는 PDF 다운로드가 제공된다

---

### US-05-4. NEWS — 보도자료 목록 탐색

**As a** 언론 관계자 또는 투자자(P4, P2),
**I want to** IR 페이지의 NEWS 섹션에서 최신 보도자료 목록을 확인하고 싶다,
**So that** 기업의 최근 동향과 이슈를 파악할 수 있다.

**Acceptance Criteria**
- [ ] 보도자료 목록이 날짜 내림차순으로 표시된다
- [ ] 각 항목에 제목, 날짜, 미리보기 텍스트(또는 썸네일)가 포함된다
- [ ] 목록 항목 클릭 시 상세 내용 또는 원문 링크로 이동한다

---

### US-05-4-S. NEWS — 페이지네이션 (Should Have)

**As a** 언론 관계자 또는 투자자(P4, P2),
**I want to** 뉴스 목록이 많을 때 페이지를 나눠 탐색하고 싶다,
**So that** 오래된 보도자료도 페이지 이동으로 찾아볼 수 있다.

## US-06. 문의하기 (/contact)

### US-06-1. 사업 문의 전송

**As a** 파트너 또는 잠재 고객(P3, P1),
**I want to** 홈페이지 문의 폼을 통해 GC메디아이에 메시지를 보내고 싶다,
**So that** 이메일을 직접 찾지 않아도 빠르게 연락을 취할 수 있다.

**Acceptance Criteria**
- [ ] 이름, 이메일, 문의 유형(사업제휴/채용/언론/기타) 선택, 제목, 내용 입력 필드가 제공된다
- [ ] 필수 항목 미입력 시 오류 메시지가 해당 필드 아래에 표시된다
- [ ] 이메일 형식이 유효하지 않으면 즉시 오류가 표시된다
- [ ] 제출 성공 시 "문의가 접수되었습니다" 안내 메시지가 표시된다

## US-08. 공통 — SEO

### US-08-1. 검색엔진 노출

**As a** 마케팅 담당자 또는 일반 방문자(내부 관점),
**I want to** GC메디아이 홈페이지가 검색 결과 상위에 노출되기를 원한다,
**So that** 잠재 고객과 투자자가 검색으로 사이트를 찾을 수 있다.

**Acceptance Criteria**
- [ ] 모든 페이지에 고유한 `<title>`과 `<meta name="description">`이 설정된다
- [ ] Open Graph 태그 (og:title, og:description, og:image)가 각 페이지에 적용된다
- [ ] `sitemap.xml`이 `/sitemap.xml` 경로에서 접근 가능하다
- [ ] `robots.txt`가 `/robots.txt` 경로에서 접근 가능하다
- [ ] 각 페이지에 `<h1>`이 하나만 존재하며 페이지 주제를 반영한다

**So that** 수신자가 링크를 클릭하기 전에 내용을 파악할 수 있다.

**Acceptance Criteria**

- [ ] og:image가 1200×630px 이상의 이미지를 참조한다
- [ ] Twitter Card 태그 (twitter:card, twitter:title, twitter:description)가 설정된다
